# ATS Resume Analyzer Lite

A free online tool built for the Digital Heroes developer trial task.

## What it does
- Takes resume text and job description text
- Calculates an ATS-style match score
- Shows matched skills
- Shows missing skills
- Gives improvement tips

## Requirements covered
- Working tool with real output
- Button labelled exactly "Built for Digital Heroes"
- Button links to https://digitalheroesco.com
- Full name and email visible on page
- Deployable on Vercel free plan
- No paid API or subscription used

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
